# Java Backend Interview Handbook

A self-contained, mobile-friendly interview handbook for **Senior Java Backend Engineer** roles — 19 chapters covering everything from Core Java to system design, with high-ROI questions, sample answers, code, and "interview gold point" follow-ups.

### 🌐 Live site: **[inter-lyart.vercel.app](https://inter-lyart.vercel.app)**

> The live site is the single-file [`handbook.html`](handbook.html) — a searchable, dark-mode-enabled reader with all chapters inlined. Each chapter also exists as its own standalone HTML file in this repo.

## ✨ Features

- **Single-file reader** — `handbook.html` works offline; just open it in any browser.
- **Dark mode** — toggle in the top bar; preference is remembered and follows your OS theme by default.
- **Mobile-adaptive** — collapsible sidebar and responsive layout.
- **Searchable** — filter chapters and sub-sections instantly.
- **400+ questions** with ROI ratings, frequency, sample answers, code snippets, and revision sheets.

## 🚀 Usage

Open [`handbook.html`](handbook.html) directly in a browser, or visit the [live site](https://inter-lyart.vercel.app).
